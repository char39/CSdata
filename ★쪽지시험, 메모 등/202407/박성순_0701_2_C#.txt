ref, out

ref : 값(스택)이 아닌 주소(힙)에 전달

ref 로 매개변수를 넘기는 경우 메서드가 해당 변수의 결과를
저장 하지 않아도 컴파일러가 아무런 경고를 하지 않고,
out 은 매개변수로 넘길 때 해당 메서드가 해당 변수의 결과를
저장 하지 않으면 컴파일러가 에러 메시지를 출력함.

ref 와 out 의 공통점 :
	출력 전용 매개변수


로컬함수 : 메서드 안에 로컬함수 선언
같은 기능이 반복적으로 연계되어 있는 경우
로컬 함수를 쓴다.

Enum 열거형 상수
	가독성을 위해서 문자는 개발자가 읽고, 컴파일러는 숫자로 읽는다.
