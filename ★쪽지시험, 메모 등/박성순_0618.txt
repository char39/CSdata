데이터 형식
1. 기본 데이터 형식
	정수, 실수
	문자[char, string]
2. 복합 데이터 형식
	클래스, 배열, 구조체

데이터 형식은 기본 데이터 형식과 복합 데이터 형식으로 분류하는 동시에
값 형식과 참조 형식으로 분류된다
값 형식은 변수가 값을 담는 데이터 형식을 말하고
참조 형식은 변수가 값 대신에 값이 있는 위치(참조)를 담는 형식

값 형식은 스택만 존재하고 중괄호는 빠져나가면 메모리에서 사라짐
참조 형식은 스택과 힙이 있고 스택에는 주소번지(힙)이 있고 힙에는 값을 저장한다.
중괄호는 빠져나가면 스택은 없어지지만 힙에 있는 값은 저장됨.

스택 메모리는 나중에 쌓인게 먼저 나가는 방식. asdf 순이면 fdsa 순으로 사라짐.
그리고 스택은 중괄호만 빠져나가면 메모리가 삭제됨.
힙은 개발자가 메모리를 할당하면, 할당한 메모리를 쓰지 않는다면
	가비지 컬렉터가 알아서 수거한다.
